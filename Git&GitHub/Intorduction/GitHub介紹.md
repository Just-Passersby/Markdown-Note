# 什麼是GitHub?
GitHub是一個目前最大的Source Code儲存平台之一，也是目前最大的Git Server，無論是Linux kernel的Source Code還是DOS的Source code都可以在GitHub上找到
2018年被微軟收購，所以目前由微軟營運，被收購後2019年便允許免費帳戶無限制開放Private Repositories
